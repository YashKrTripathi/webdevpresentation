CSR (Client-Side Rendering)

The browser downloads a minimal HTML file and a large JS bundle. The JS then generates the entire page. Fast transitions, slow first load.

bolt High Interactivity

SSR + Hydration (Server-Side Rendering)

The server generates the full HTML on each request. Hydration makes it interactive after it reaches the browser. Instant first load, best for SEO.
